# Тестовое задание

**Особенности решения:**
 * Java 21
 * Сборка через Maven 3.9.8
 * Использован `maven-jar-plugin` версии 3.4.1
 * Ключ -f имеет приоритет над -s
 * Список файлов на вход подаётся в конце списка параметров
 * Файлы пишутся и читаются построчно
   * Это экономит оперативную память в случае больших файлов или большого количества файлов
   * В случае экстренной остановки программы у пользователя будет частичный результат выполнения
 * Весь вывод пользователю имеет английский язык
 * Если вы применяете ключ -p и указываете префикс с пробелами, то префикс должен быть окружён кавычками (")
 
Пример запуска программы:
`java -jar data_transfer-1.0.jar -a -p results- input.txt`
